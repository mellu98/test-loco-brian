# test-loco-brian

Questo repository include un nuovo ambiente di sviluppo pronto all'uso.

## File ambiente
- `.env.example`: template di riferimento.
- `.env.development`: configurazione di sviluppo locale.

## Utilizzo rapido
1. Copia il file ambiente desiderato:
   ```bash
   cp .env.development .env
   ```
2. Personalizza i valori in base al tuo setup locale.
